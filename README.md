# osu! 镜像下载客户端
## 主要特性
1. 支持选择安装路径和分支。
2. 支持防篡改及文件完整性校验，并据此实现了增量更新。
3. 支持多 Mirror 和第三方 Mirror，可通过文件完整性检查防止第三方 Mirror 篡改，并可自动检测延迟进行升序排列 (由低到高)。
4. 服务端无需人工干预，可在短时间内镜像最新 osu! 版本 (在新版本尚未镜像成功前将提供上一版本)。
